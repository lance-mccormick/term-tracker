D424 Software Engineering Capstone

Features:

  • View and manage academic terms and associated courses
  • Track assessments with due dates and receive local notifications
  • SQLite database integration for offline storage
  • Clean and responsive mobile interface

This mobile application was developed as a capstone project for the Software Engineering program at WGU. The app is designed to help WGU students manage their academic progress by tracking terms, courses, and assessments. Built using .NET MAUI and SQLite, the application supports adding, editing, and deleting terms and courses, setting reminders for assessment deadlines, and sharing course notes.
